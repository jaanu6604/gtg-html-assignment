# GTG HTML Developer Assignment

This repository contains a responsive frontend implementation based on the provided Figma design.

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)

## Features
- Responsive layout using Flexbox
- Accordion interaction for product collections
- Animated percentage counters on scroll
- Clean and modular code structure
- Lazy loading for images

## How to Run
- Open `index.html` using Live Server in VS Code

## Notes
- Implemented according to the given Figma design
- No external frameworks or libraries used
